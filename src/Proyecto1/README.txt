*Para comenzar el juego:

1) Se compilan todos los archivos
   
   Estos estan enlazados con un package llamado "practica2"
   el nombre no se cambio por falta de tiempo 

2) Ejecutar el archivo: UsaJuego.java 

   Este archivo inicia el juego



*¿Como jugar?

   -Al principio te pedira ingresar el numero de jugadores,
    este tiene que ser mayor o igual a 15.
   -Luego te mostrara una pantalla con el personaje asignado
    de manera aleatoria a cada jugador 
   -En el resto del juego unicamente tienes que elegir
    una de las opciones que te muestre

*Precaucion:
    
    -Por falta de tiempo no se usaron excepciones, 
     por lo tanto intenta ingresar los nombres correctamente,
     de manera identica como se ingresaron al registrar a los jugadore, 
     sin agregar espacios ni otro caracter que no se haya registrado, para
     poder disfrutar de una buena experiencia.

    -El juego esta implentado para linux, ya que se utiliza un metodo que limpia
     el terminal,(no estamos seguros si funciona en otro sistema operativo).


¡A jugar!
